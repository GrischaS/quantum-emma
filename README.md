## 🚀 Quick Start

### Web App (Live)
```
https://quantum-emma-app.base44.app
```

### Windows Desktop
```bash
npm install && npm run build:win
# → dist/QEMMA-Setup-v4.1.0.exe (~148 MB)
```

### Android APK
```bash
cd android && npm install
eas build --platform android --profile preview
# → QEMMA-v4.1.0.apk (~48 MB)
```

### Smart Contracts
```bash
cd contracts
npx hardhat compile
npx hardhat run deploy/deploy.js --network sepolia
```
